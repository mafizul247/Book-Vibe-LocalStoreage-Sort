# 📚 Book Vibe

Book Vibe is a modern React-based book management application where users can explore books, save them to local storage, and sort items with a clean and responsive interface.

🔗 **Live Demo:** https://sensational-kulfi-b3c9ed.netlify.app/  
📂 **GitHub Repository:** https://github.com/mafizul247/Book-Vibe-LocalStoreage-Sort.git  

---

## 🚀 Features
- 📱 Fully responsive navbar
- 📚 Display and manage book items
- 💾 Add and retrieve, delte data from Local Storage
- 🔄 Sort books/items dynamically
- 🔔 Beautiful alert notifications using SweetAlert
- 🎨 Modern UI with Tailwind CSS & DaisyUI
- ⚡ Fast and interactive React experience

---

## 🛠️ Tech Stack
- **React JS** – Frontend library
- **Tailwind CSS** – Utility-first CSS framework
- **DaisyUI** – Tailwind CSS component library
- **SweetAlert2** – Beautiful popup alerts
- **JavaScript (ES6+)**
